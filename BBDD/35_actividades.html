3.5. Actividades

Actividad 3.6.

Crea una base de datos nueva llamada Empresa en donde crearás tres tablas, las necesarias para implementar en LibreOffice Base el problema de la práctica guiada del apartado 2.3.

Piensa qué tipo de datos debes de elegir para cada campo así como la longitud idónea para los datos que se deban introducir.

Asigna la clave principal al campo adecuado y crea las relaciones necesarias.

Actividad 3.7.

Crea una base de datos definiendo las tablas necesarias, creando las claves principales en los campos correspondinetes y estableciendo las relaciones necesarias para el caso de la actividad 2.2. Entrenadores de Fútbol. Guárdala con el nombre Entrenadores de Futbol.

A continuación introduce datos en las diferentes tablas (por lo menos 3 registros por tabla) y comprueba que todo funciona correctamente.

Actividad 3.8.

Crea una base de datos nueva llamada Vuelta Ciclista en donde crearás las 6 tablas necesarias (5 de entidades y 1 de la relación M:N entre maillot y etapa) para implementar en LibreOffice Base el problema de la Actividad 4 del apartado 2.5. Creando las claves principales en los campos correspondinetes y establece las relaciones necesarias 

Guárdala con el nombre Vuelta Ciclista.

A continuación introduce datos en las diferentes tablas (por lo menos 3 registros por tabla) y comprueba que todo funciona correctamente.
