4.1. Creación de Consultas

Formas de crear una consulta en OpenOffice Base

Existen tres formas de crear consultas en LibreOffice Base según las características de cada proyecto:

  • Podrás crear una consulta en la vista Diseño.
  • Utilizar el asistente para consultas.
  • O trabajar con el lenguaje de consultas estructurado SQL, si bien es cierto que esta última opción sólo es recomendable para personas con un alto grado de experiencia en bases de datos.

En la siguiente figura se pueden observar las tres posibilidades desde el programa LibreOffice Base.

 

 

Al principio, para consultas sencillas, lo recomendable es crear una consulta utilizando el asistente para consultas. Una vez se ha practicado con varias consultas, se podría trabajar con la vista diseño.

Lo más complicado (debido a que es necesario aprenderse un lenguaje de consultas) es utilizar SQL.
