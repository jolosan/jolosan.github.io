4.5. Actividades

Actividad 4.2.

Sobre la base de datos empresa.odb realiza las siguientes consultas:

1. Crea una consulta con el nombre Cursos que realizan los departamentos en la que aparezcan el nombre del curso y el nombre del departamento que lo realiza.

2. Crea una consulta con el nombre Cursos realizados por Loli. Usa únicamente las tablas Curso, Realiza y Trabajador.

3. Crea una consulta con el nombre Horas de los cursos de  Juan que sume las horas de los cursos del trabajador Juan.

4.  Crea una consulta con el nombre Cursos de mas de 20 horas que obtenga los cursos cuya duración es mayor de 20 horas.

 

Actividad 4.3.

Disponemos de una base de datos que lleva el registro de los libros que hay en una biblioteca. Actualmente cuenta con 1840 títulos, y de cada título se almacena:

  • nº registro
  • título
  • autor
  • editorial
  • año de adquisición

Sobre la base de datos Biblioteca.odb realiza las siguientes consultas:

1. Crea una consulta con el nombre Libros comprados en 1989 de la editorial Anaya que obtenga dichos libros.

2. Crea una consulta con el nombre Cantidad de libros comprados en 1989 de la editorial Anaya que obtenga dicha cantidad libros (el resultado es 15).

3. Crea una consulta con el nombre Libros sin autor y de la editorial Castalia que muestre el titulo de estos líbros (el nº de registros es 2).

 

Actividad 4.4.

Abre la base de datos Animales.odb.

1. Crea una una consulta con el nombre especies por clase que  cuente cuantos animaleshay por cada clase.

2. Crea una vista con el nombre especies ovíparas que sólo incluya a esas especies.

3. Crea una vista con el nombre especies no ovíparas que sólo incluya a esas especies.

4. Crea una consulta con el nombre cantidad de especies de roedores que cuente cuantas especies hay de la orden Roedores.
